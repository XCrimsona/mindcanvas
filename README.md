# MindCanvas

A full-stack productivity app built with the MERN stack. MindCanvas helps users organize information, tasks, and ideas with a visually appealing interface, secure authentication, and powerful features.

## Version

**Current local version:** 2.1.0  
_Active upgrade in progress_

_Remote authentication is being improved. Local authentication (with Bcrypt.js) is implemented but still undergoing development._

## Purpose

MindCanvas enables users to capture, structure, and visualize their thoughts, tasks, and projects efficiently. This project serves as a practical and experimental showcase for SDE skills and full-stack best practices.

## Features

- **Secure authentication:** User account creation and login powered by Bcrypt.js (MERN stack).
- **Task/idea management:** Create, edit, delete, and group tasks, notes, and concepts.
- **Visual dashboard:** Modern, responsive UI for organizing and exploring information.
- **Project and category organization:** Group related notes, tasks, or ideas (if implemented).
- **Personalization:** User-specific boards and customizable workspace.
- **Reliable backend:** Built with Node.js, Express, MongoDB, and strong TypeScript typing.
- **Responsive design:** Uses SCSS and CSS for smooth experience across devices.
- **In-progress remote features:** Authentication and connectivity continue to improve.
